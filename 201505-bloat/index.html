Chromium binary size, May 2015

This is a map of Chromium binary, showing how many bytes do our various bits of code contribute to the final executable. It's built using webtreemap and bloat.

It's 100MB of json file, be patient-- a static preview image (in grayscale) is being shown while it's loaded. Once the entire map is loaded, click on a box to zoom in, click on the outermost box to zoom out.

References:

  • Chromium binary size, Sep 2010 (by Evan Martin)
  • Chromium binary size, Jan 2014 (by Nico Weber)

It looks the Chromium binary size is now about 3 times larger than it was 5 years ago.
